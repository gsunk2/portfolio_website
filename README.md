## 🚀 Deployment
To deploy on GitHub Pages:
1. Push the repo to GitHub
2. Go to **Settings > Pages**
3. Select branch `main` and root folder `/`
4. Save → Your portfolio will be live at:
   https://your-username.github.io/portfolio-website
